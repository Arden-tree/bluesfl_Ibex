# LiK

```bash
cd /home/lzz/dac26/LiK_ibex
# 1. extract comments from rtl source code
python scripts/01_extract_comments.py -i $IBEX_HOME_PATH/rtl -o ./ibex_info
# 2. generate description using gpt-4
python scripts/02_llm_summary.py -i ./ibex_info -o ./ibex_description
# 3. generate bugs info for dataset
python scripts/03_copy_bugs.py -i /home/lzz/dac26/hdl_fl_data/dataset -o ./dataset_bugs
# 4. run inference
python lik_inference_ibex.py -i ./ibex_info -b ./dataset_bugs -o ./lik_results -p lik_res
```

result will be in folder `lik_results`

collect all results to one. Note that `lik_res_merged_line_results.json` only contains lineno.

```bash
python scripts/collect_loc_results.py \
  --root=/home/lzz/dac26/lik_server_results/LiK_ibex_1107/lik_results \
  --output=./results/lik/lik_res_merged_line_results.json \
  --prefix=lik_res
```

Then use `src/bin/l2b.rs` to convert lineno to block id.
